Welcome to Matrix 3, a JavaScript-based falling-sand game.

Tips:
  • The world is infinite. Use the arrow keys to navigate. Hold shift to go faster.
  • Use left/right click to draw.
  • Use 0-9 to change draw size.
  • The game runs really poorly on Chrome. Try a different browser if you want better performance.
